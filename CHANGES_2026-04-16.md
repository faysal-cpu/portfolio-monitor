## Changes Made

### 1. Fixed Duplicate Email Issue

**Problem:** User receiving two emails daily - one at 3 AM and one at 7 AM

**Root Cause:** Multiple instances running (Railway in UTC + local instance)

**Fix:**
- Added instance identification logging (hostname) to `run_portfolio_analysis()`
- Added comprehensive timezone warning in `schedule_job()` docstring
- Added debugging logs showing which instance is sending emails
- Created `DUPLICATE_EMAIL_FIX.md` with troubleshooting guide

**Files Modified:**
- `main.py` - Lines ~1215-1219, ~1319-1340

**What Changed:**
```python

Time: 2026-04-16 07:12:42
```

This helps identify if Railway (UTC) or local instance sent the email.

### 2. Removed Preamble Text from Macro Context

**Problem:** Email included unwanted text: "Here are the 3 most important macro and geopolitical factors a Canadian retail investor needs to know today, April 16, 2026:"

**Fix:** Updated Claude prompt to output ONLY bullet points without any introductory text

**Files Modified:**
- `main.py` - Lines ~165-173

**Old Prompt:**
```python
prompt = f"""Today is {date_str}. Search for today's most important market news 
and macro developments. In 3 bullet points, give the most important macro and 
geopolitical factors a Canadian retail investor should know TODAY that could 
affect North American and global markets. Be specific — mention actual events, 
not generic risks."""
```

**New Prompt:**
```python
prompt = f"""Today is {date_str}. Search for today's most important market news 
and macro developments.

Output ONLY 3 bullet points (using emoji bullets like 🛢️, 🏛️, 📉) covering the 
most important macro and geopolitical factors a Canadian retail investor should 
know TODAY.

Do NOT include any introductory text, preamble, or sentences before the bullet points.
Do NOT write "Here are..." or "Today..." or any other introduction.
Start IMMEDIATELY with the first bullet point.

Be specific — mention actual events, not generic risks."""
```
